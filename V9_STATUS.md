# DroidForge V9 Android-only status

- Android is the only target retained in this source tree.
- JDK 17 is the active implementation target.
- JDK 21 and JDK 24 are shown as Coming Soon.
- Desktop/Linux JDK and SDK downloads are blocked.
- A verified Android ARM64 Java runtime and Android ARM64 SDK tool bundle are still required before installation can be functional.
- Installed/Ready states require executable verification rather than file existence only.
